# 大富翁游戏 - 安装说明

## 🚀 快速开始

### 环境要求
- **Android设备**: Android 7.0 (API 24) 或更高版本
- **存储空间**: 至少 20MB 可用空间
- **网络**: 不需要网络连接（离线游戏）

### 安装方式

#### 方式一：直接安装APK（推荐）
1. 下载 `app-debug.apk` 文件到你的Android设备
2. 在设备上启用"未知来源应用安装"：
   - 打开 **设置** → **安全** → 开启 **未知来源**
   - 或者 **设置** → **应用和通知** → **特殊应用访问** → **安装未知应用**
3. 用文件管理器找到APK文件，点击安装
4. 完成后即可在应用列表中找到"大富翁游戏"

#### 方式二：开发者构建
如果你是开发者，可以从源码构建：

```bash
# 克隆项目
git clone <项目地址>
cd MonopolyGame

# 构建APK
./build.sh

# APK将生成在: app/build/outputs/apk/debug/app-debug.apk
```

### 设备要求

#### 最低配置
- **CPU**: 任何ARM或x86处理器
- **内存**: 1GB RAM
- **存储**: 20MB可用空间
- **屏幕**: 480x800像素（推荐横屏游戏）

#### 推荐配置
- **CPU**: 四核1.4GHz或更高
- **内存**: 2GB RAM或更高
- **存储**: 100MB可用空间
- **屏幕**: 1080x1920像素或更高（平板体验更佳）

### 游戏控制

#### 基本操作
- **掷骰子**: 点击"掷骰子"按钮
- **购买房产**: 停在无主房产时点击"购买房产"
- **建造建筑**: 在自己的房产上点击"建造房屋/酒店"
- **结束回合**: 完成操作后点击"结束回合"

#### 界面说明
- **中央棋盘**: 显示游戏棋盘和玩家位置
- **左侧面板**: 显示所有玩家信息和状态
- **右侧面板**: 显示当前可执行的操作
- **底部控制**: 游戏主要控制按钮

### 故障排除

#### 常见问题

**Q: 安装时提示"解析包时出现问题"**
A: 请确保：
- APK文件完整下载，没有损坏
- Android版本在7.0以上
- 存储空间充足

**Q: 安装后无法打开**
A: 请尝试：
- 重启设备后再试
- 清除应用缓存
- 重新安装应用

**Q: 游戏运行卡顿**
A: 建议：
- 关闭后台不必要的应用
- 重启设备释放内存
- 在设置中关闭动画效果

**Q: 屏幕显示不全**
A: 请：
- 切换到横屏模式（推荐）
- 调整设备显示缩放比例
- 确保设备分辨率在支持范围内

#### 性能优化建议

1. **横屏游戏**: 游戏专为横屏设计，横屏体验最佳
2. **关闭后台**: 游戏运行时关闭不必要的后台应用
3. **充足电量**: 确保设备电量充足，避免低电量模式影响性能
4. **稳定网络**: 虽然游戏离线运行，但安装时需要稳定的下载环境

### 卸载方法

如需卸载游戏：
1. 长按应用图标
2. 选择"卸载"或拖动到垃圾桶
3. 确认卸载

或者：
1. 进入 **设置** → **应用管理**
2. 找到"大富翁游戏"
3. 点击"卸载"
