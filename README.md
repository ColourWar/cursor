# 大富翁游戏 (Monopoly Game)

一个功能完整的大富翁Android游戏，采用现代化设计和流畅的用户体验。

## 游戏特色

### 🎮 核心游戏功能
- **完整的大富翁游戏体验** - 掷骰子、移动、购买房产、收取租金
- **多人游戏支持** - 2-4名玩家同时游戏
- **台湾主题棋盘** - 以台湾各县市为房产名称
- **房屋建造系统** - 支持建造房屋和酒店提高租金
- **监狱机制** - 完整的监狱和出狱系统
- **机会/命运卡片** - 随机事件增加游戏趣味性

### 🎨 精美界面设计
- **横屏游戏体验** - 专为平板和手机横屏优化
- **自定义棋盘视图** - 手绘式棋盘，色彩丰富
- **动画效果** - 掷骰子动画、平滑过渡效果
- **玩家状态显示** - 实时显示玩家资金、房产、状态
- **直观的操作界面** - 简洁易用的按钮和面板设计

### 🏠 房产系统
- **40个棋盘格子** - 包含房产、特殊格子、机会卡等
- **9种房产颜色** - 不同颜色代表不同价值级别
- **建筑升级** - 房屋→酒店的升级路径
- **租金计算** - 根据建筑等级动态计算租金
- **所有权管理** - 完整的房产购买和管理系统

## 游戏规则

### 基础规则
1. **游戏开始** - 每位玩家初始资金$1500，从起点开始
2. **回合进行** - 掷骰子→移动→执行格子效果→结束回合
3. **过起点奖励** - 每次经过起点获得$200
4. **胜利条件** - 其他玩家全部破产时获胜

### 房产规则
- **购买房产** - 停在无主房产上可选择购买
- **支付租金** - 停在他人房产上需支付租金
- **建造建筑** - 拥有房产后可建造房屋或酒店
- **租金计算**：
  - 基础租金：房产基础租金
  - 有房屋：基础租金 × 2
  - 有酒店：基础租金 × 5

### 特殊格子
- **起点** - 经过获得$200奖励
- **监狱** - 只是参观，无特殊效果
- **进监狱** - 直接进入监狱
- **免费停车** - 无特殊效果，安全停靠
- **税收** - 缴纳税款给银行
- **机会/命运** - 抽取随机卡片

## 安装说明

### 方式一：源码编译
1. 克隆项目到本地
```bash
git clone <project-url>
cd MonopolyGame
```

2. 使用Android Studio打开项目

3. 连接Android设备或启动模拟器

4. 点击运行按钮进行安装

### 方式二：APK安装
1. 下载编译好的APK文件
2. 在设备上启用"未知来源应用安装"
3. 安装APK文件
